# Customer Churn ML Project

## Project Overview

This project focuses on predicting customer churn using Machine Learning and Data Analytics techniques. The analysis helps businesses identify customers who are likely to leave and supports better decision-making for customer retention.

---

## Technologies Used

* Python
* Machine Learning
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* SQL
* Jupyter Notebook

---

## Project Files

* `customer churn analysis.ipynb` – Main Jupyter Notebook
* `3_create_analytics_table.sql` – SQL queries used for analysis
* `Master_table.csv` – Processed dataset
* `Customer Churn Analysis Report.pptx` – Project presentation
* `Data Sets/` – Raw datasets used in the project

---

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Building
6. Customer Churn Prediction
7. Result Analysis

---

## Machine Learning Models Used

* Logistic Regression
* Random Forest
* Decision Tree

---

## Key Insights

* Identified important factors affecting customer churn
* Built predictive models for churn analysis
* Improved understanding of customer behavior patterns

---

## Conclusion

This project demonstrates how Machine Learning and Data Analytics can help businesses predict customer churn and improve customer retention strategies.
